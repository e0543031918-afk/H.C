# STUDIO ✂️ — אפליקציה למעצבי שער

אפליקציית ניהול פונקציונלית לסטייליסטים ומעצבי שער.
קובץ HTML יחיד, ללא תלויות חיצוניות — נפתח בכל דפדפן. כל הכפתורים עובדים,
והנתונים נשמרים בדפדפן (localStorage).

## ✨ מסכים

| מסך | תיאור |
|------|--------|
| 🏠 **בית** | סיכום הכנסות יומי, סטטיסטיקה מהירה והתורים הבאים |
| 📅 **יומן** | תצוגת שבוע, ציר זמן יומי וקביעת תור חדש |
| 🖼️ **פורטפוליו** | גלריית עבודות עם סינון לפי קטגוריה |
| 📊 **העסק** | הכנסות/הוצאות, גרף 6 חודשים והתראות מלאי |
| 💇 **התאמת סגנון** | בחירת מבנה פנים והמלצות תספורת/צבע עם אחוזי התאמה |

## ⚙️ פעולות פעילות

הוספה/מחיקה/סימון תורים · ניווט יומן לפי יום · סינון פורטפוליו והוספת עבודות ·
ניהול מלאי (הוספה ועדכון כמויות) · התאמת המלצות לפי מבנה פנים · 3 ערכות עיצוב.
כל שינוי נשמר אוטומטית בדפדפן.

## 🎨 3 ערכות עיצוב

- **מודרני ונקי** — שחור-לבן, מינימליסטי
- **צבעוני ותוסס** — גרדיאנט ורוד-סגול
- **יוקרתי** — זהב על רקע כהה

מחליפים בין הערכות בכפתורים שמעל מסגרת הטלפון.

## 🚀 הפעלה

```bash
# פשוט פותחים את הקובץ בדפדפן
open index.html
```

או דרך שרת מקומי:

```bash
python3 -m http.server 8000
# גשו אל http://localhost:8000
```

## 🛠️ טכנולוגיה

HTML + CSS + JavaScript נטו (Vanilla). תמיכה מלאה ב-RTL. ללא build, ללא npm.
